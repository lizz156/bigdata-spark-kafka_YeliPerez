# Análisis de datos con Spark y Kafka

##  Descripción

En este proyecto se realiza el análisis de datos utilizando dos enfoques:

- Procesamiento por lotes (batch) con Apache Spark  
- Procesamiento en tiempo real con Kafka y Spark Streaming  

---

##  Entorno de trabajo

Todos los comandos fueron ejecutados en una máquina virtual previamente configurada con Hadoop y Spark.

---

##  Procesamiento por lotes (Batch)

Archivo:
- batch_analysis.py

Ejecución:

```bash
spark-submit batch_analysis.py
